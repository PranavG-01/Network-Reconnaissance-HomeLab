# Network-Recon-HomeLab
## Objective: 
Utilize nmap to perform network reconnaissance in a virtual environment - identifying live hosts, port scanning, and security auditing

## Lab Setup:
Virtual Environment (VirtualBox)
  Attacker: Kali Linux VM
  Target: Windows 10 VM
Network: NAT (default mode)
Tools: Nmap

## Action Steps:
1) Discovered live hosts with ping scanning
2) Practiced OS and version detection
3) Analyzed the different states of TCP ports and running services
4) Analyzed exposed services for potential risk
